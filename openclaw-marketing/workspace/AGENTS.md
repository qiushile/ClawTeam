# 市场营销部专业 Agent 库

你可以根据任务需求，调用以下专业角色来协助完成工作。这些 Agent 详细的指令位于 `workspace/agents/` 目录中。

## 内容与增长
- **增长黑客 (growth-hacker)**: 测试裂变机制以超低成本完成病毒性增量增长。
- **内容创作者 (content-creator)**: 各类行销内容的核心文字输出、宣发计划表的整合及内容再利用流转。
- **图书联合撰稿人 (book-co-author)**: 辅助知识萃取与图书合作写作出版系统化内容包装。
- **播客策略师 (podcast-strategist)**: 提供基于主流播客的受众培养策略、叙事引导与商业化连接。
- **短视频剪辑教练 (short-video-editing-coach)**: 输出各大剪辑软件平台中高效产出爆款成片的指导逻辑流。
- **图文增长引擎 (carousel-growth-engine)**: 洞悉图文类（尤其是多页旋转式）爆款结构及转化促活的脚本分发机制。

## 搜索引擎与分销
- **SEO 专家 (seo-specialist)**: 技术型通用搜索引擎优化排布规则（包含链路、元标记优化）。
- **百度 SEO 专家 (baidu-seo-specialist)**: 适配特定中国本土市场规则的主动与被动展现优化流量抓取。
- **应用商店优化师 (ASO) (app-store-optimizer)**: 根据不同商店转化池深度制定 ASO 出价策略与吸睛主图测试规划。

## 电商及私域转化阵地
- **跨境电商专家 (cross-border-e-commerce-specialist)**: 在品牌出海及跨平台跨境业务矩阵化部署下优化转化。
- **国内电商运营专家 (china-e-commerce-operator)**: 国内顶级综合电商平台活动期与平销期结合的操盘管理。
- **直播电商教练 (livestream-commerce-coach)**: 优化直播间的全盘控场话术、排品推流和连麦互动情绪价值。
- **私域运营专家 (private-domain-operator)**: 从零到一建立且长续运营企微/微信私域生态并优化复购流。

## 国际社媒生态平台
- **TikTok 策略师 (tiktok-strategist)**: 洞悉推荐主页的内容生态与流量模型以打造全球话题热度。
- **Instagram 策展人 (instagram-curator)**: 引导如何利用极致美学的矩阵图与短动态激发强互动表现。
- **LinkedIn 内容创作者 (linkedin-content-creator)**: 定位个人或品牌的商业见解及企业白皮书以触达精准 B2B 受众。
- **Twitter 互动专家 (twitter-engager)**: 以轻量快文与强观点引发互动讨论并实时跟进话题链。
- **Reddit 社区构建者 (reddit-community-builder)**: 运用本土亚文化深入沟通提供社区认同式价值反馈引导。

## 国内社媒生态平台
- **抖音策略师 (douyin-strategist)**: 深入研究快频推送逻辑及高转化带货和爆款达人的运作核心。
- **B站内容策略师 (bilibili-content-strategist)**: 精英化长视频脚本和独特的弹幕互动氛围生态运营指南。
- **快手策略师 (kuaishou-strategist)**: 下沉核心群体的强人格背书与私域直播流量商业矩阵赋能。
- **小红书专家 (xiaohongshu-specialist)**: 运用趋势关键词标签策略，提供种草美学灵感和避雷指南写法。
- **知乎策略师 (zhihu-strategist)**: 靠权威科普式答主定位以高信服度转化优质的长尾高留存私域种子。
- **微博策略师 (weibo-strategist)**: 获取公共空间瞬时流量引爆事件行销热点的矩阵公排。
- **微信公众号管理员 (wechat-official-account-manager)**: 以提供持续沉淀的高价值信息为主干进行私域大图文触达。
