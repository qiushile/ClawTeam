| **安全管理人员** | 录用、培训、考核 | 背景调查、定期培训 |
| **安全建设管理** | 定级、设计、实施 | 按等保要求建设 |
| **安全运维管理** | 运维、监控、应急 | 定期检测、应急预案 |

## 四、数据合规

### 4.1 数据分类

| 数据类型 | 级别 | 说明 |
|---------|------|------|
| **个人信息** | 一般 | 姓名、手机号、邮箱 |
| **敏感个人信息** | 敏感 | 身份证、人脸、支付信息 |
| **企业数据** | 一般 | 企业名称、统一信用代码 |
| **业务数据** | 重要 | 用户训练数据、代码、模型 |
| **运营数据** | 一般 | 日志、监控数据 |

---

### 4.2 个人信息保护

| 要求 | 说明 | 我们的措施 |
|-----|------|-----------|
| **告知同意** | 明确告知收集目的 | 隐私政策、用户协议 |
| **最小必要** | 仅收集必要信息 | 信息收集清单 |
| **目的限制** | 仅用于约定目的 | 信息使用规范 |
| **安全保障** | 采取安全措施 | 加密、权限控制 |
| **权利保障** | 保障用户权利 | 访问、更正、删除权 |

---

### 4.3 数据出境

| 情形 | 要求 | 我们现状 |
|-----|------|---------|
| **数据境内存储** | 个人信息境内存储 | ☐ 承诺遵守 |
| **数据出境评估** | 出境需通过安全评估 | ☐ 暂无出境需求 |
| **跨境传输** | 标准合同/认证 | ☐ 暂无需求 |

**建议：** 原则上数据不出境，确需出境时进行安全评估

---

### 4.4 数据处理协议

**与企业客户签署数据处理协议，明确：**

| 内容 | 说明 |
|-----|------|
| **数据权属** | 客户数据归客户所有 |
| **处理目的** | 仅用于提供服务 |
| **处理方式** | 按客户指示处理 |
| **安全措施** | 采取适当安全措施 |
| **数据删除** | 服务终止后删除数据 |
| **审计权利** | 客户有权审计合规情况 |

---

### 4.5 数据合规风险

| 风险 | 说明 | 处罚 |
|-----|------|------|
| **未告知同意** | 未明确告知收集使用规则 | 责令改正、警告 |
| **超范围收集** | 收集与服务无关信息 | 责令改正、罚款 |
| **数据泄露** | 未采取安全措施导致泄露 | 罚款、停业整顿 |
| **违规出境** | 未经评估出境 | 罚款、刑事责任 |

---

### 4.6 数据合规建议

| 建议 | 说明 |
|-----|------|
| **制度先行** | 制定数据管理制度 |
| **技术保障** | 加密、脱敏、权限控制 |
| **人员培训** | 定期数据安全培训 |
| **应急预案** | 制定数据泄露应急预案 |
| **定期审计** | 定期自查合规情况 |
