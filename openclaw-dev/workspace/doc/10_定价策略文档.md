## 一、定价原则与目标

### 1.1 定价原则

| 原则 | 说明 |
|-----|------|
| **竞争力优先** | 比主流云厂商低 30-40%，比 AutoDL 高 10-20% |
| **成本覆盖** | 确保毛利率>40%（成熟期） |
| **简单透明** | 价格清晰，无隐藏费用 |
| **灵活多样** | 多种计费模式满足不同需求 |
| **动态调整** | 根据供需和成本动态调整 |

---

### 1.2 定价目标

| 目标 | 说明 | 指标 |
|-----|------|------|
| **市场渗透** | 快速获客，建立规模 | 6 个月内 50 家付费客户 |

| **利润优化** | 平衡收入和毛利 | 毛利率 40%+ |
| **客户留存** | 长期合同锁定客户 | 年留存率 60%+ |

## 八、价格调整机制

### 8.1 调价触发条件

| 条件 | 说明 | 调整幅度 |
|-----|------|---------|
| **GPU 采购成本变化>10%** | 硬件涨价/跌价 | ±5-10% |
| **IDC 成本变化>10%** | 电费/机柜涨价 | ±3-5% |
| **利用率<50% 持续 3 个月** | 资源闲置 | -5-10% |
| **利用率>80% 持续 3 个月** | 资源紧张 | +5-10% |
| **竞品调价** | 主要竞品价格变化 | 跟随调整 |

---

### 8.2 调价流程

```
1. 财务部提出调价建议（数据支撑）
         │
2. CEO 审批（考虑市场影响）
         │
3. 提前 7 天通知现有客户
         │
4. 官网更新价格
         │
5. 新价格生效
```

---

### A. 价格计算器公式

```
实例费用 = GPU 单价 × GPU 数量 × 使用时长 × 折扣

存储费用 = Σ(存储类型容量 × 单价)

网络费用 = 出站流量 × 单价（超出免费额度部分）

总费用 = 实例费用 + 存储费用 + 网络费用 + 增值服务
```
