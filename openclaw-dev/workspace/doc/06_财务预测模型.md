### 8.2 核心公式说明

**收入计算：**
```
月收入 = GPU 卡数 × 利用率 × 单价 × 24 小时 × 30 天
年收入 = SUM(月度收入)
```

**成本计算：**
```
硬件折旧 = 采购成本 × (1-残值率) / 折旧年限
IDC 成本 = 机柜数 × (机柜单价 + 电费 + 带宽) × 12
人力成本 = 人数 × 人均成本
```

**利润计算：**
```
毛利润 = 营业收入 - 营业成本
EBITDA = 净利润 + 折旧摊销 + 财务费用
净利润 = 税前利润 - 所得税
```

**现金流计算：**
```
经营现金流 = 净利润 + 折旧摊销 - 营运资金变动
投资现金流 = -硬件采购 - 其他投资
融资现金流 = 股权融资 + 债权融资
期末现金 = 期初现金 + 经营现金流 + 投资现金流 + 融资现金流
```

---

### 8.3 关键指标计算

**回本周期：**
```
回本周期 = 硬件投资 / (月收入 - 月运营成本)
```

**毛利率：**
```
毛利率 = 毛利润 / 营业收入 × 100%
```

**EBITDA 利润率：**
```
EBITDA 利润率 = EBITDA / 营业收入 × 100%
```

**客户 LTV：**
```
LTV = 客单价 × 平均留存月数 × 毛利率
```

**获客 ROI：**
```
ROI = LTV / CAC
```
