## 十、核心公式说明

### 10.1 收入计算公式

```excel
GPU 收入 = GPU 卡数 × 利用率 × 单价 × 8760 小时 / 10000
存储收入 = 存储容量 (TB) × 1024 × 单价 (元/GB/月) × 12 / 10000
总收入 = GPU 收入 + 存储收入 + 增值服务收入
```

---

### 10.2 成本计算公式

```excel
硬件折旧 = 硬件采购成本 / 5 年
IDC 成本 = (机柜单价 + 电费 + 带宽) × 机柜数 × 12
人力成本 = 人数 × 人均成本
总成本 = 硬件折旧 + IDC 成本 + 人力成本
```

---

### 10.3 利润计算公式

```excel
毛利润 = 营业收入 - 营业成本
毛利率 = 毛利润 / 营业收入
税前利润 = 毛利润 - 费用
所得税 = IF(税前利润>0, 税前利润×15%, 0)
净利润 = 税前利润 - 所得税
```

---

### 10.4 现金流计算公式

```excel
经营现金流 = 净利润 + 折旧 - 营运资金变动
投资现金流 = -硬件采购 - 其他投资
融资现金流 = 股权融资 + 债权融资
期末现金 = 期初现金 + 经营现金流 + 投资现金流 + 融资现金流
```

---

### 10.5 关键指标计算公式

```excel
回本周期 = 硬件投资 / (月收入 - 月运营成本)
LTV = 客单价 × 平均留存月数 × 毛利率
CAC = 获客总成本 / 获客数
LTV/CAC = LTV / CAC
```
