---

### 2.3 Logo 最小尺寸

| 媒介 | 最小尺寸 | 说明 |
|-----|---------|------|
| **印刷品** | 宽度 20mm | 保证清晰度 |
| **屏幕显示** | 宽度 80px | 保证可读性 |
| **favicon** | 16×16px | 仅图形部分 |
| **APP 图标** | 512×512px | 完整图形 |

---

### 2.4 Logo 安全区域

```
        ┌───────────────────────────┐
        │                           │
        │    ┌─────────────┐        │
        │    │             │        │
        │    │    LOGO     │   = x  │
  = x   │    │             │        │
        │    └─────────────┘        │
        │                           │
        │         = x               │
        └───────────────────────────┘

安全区域：Logo 周围至少保留 x 高度的空白
x = Logo 高度的 1/2
```

---

### 2.5 Logo 错误用法

| 错误示例 | 说明 |
|---------|------|
| ❌ 拉伸变形 | 保持原始比例 |
| ❌ 更改颜色 | 使用标准色值 |
| ❌ 添加效果 | 不加阴影、渐变 |
| ❌ 背景杂乱 | 确保足够对比度 |
| ❌ 旋转使用 | 保持水平方向 |
| ❌ 拆分使用 | 保持完整组合 |

## 四、字体规范

### 4.1 中文字体

| 用途 | 字体 | 备用字体 | 说明 |
|-----|------|---------|------|
| **标题** | 思源黑体 Bold | 微软雅黑 Bold | 醒目、有力 |
| **正文** | 思源黑体 Regular | 微软雅黑 | 清晰易读 |
| **代码** | 思源等宽 | Consolas | 等宽字体 |
| **数字** | DIN Alternate | Arial | 现代感 |

---

### 4.2 英文字体

| 用途 | 字体 | 备用字体 | 说明 |
|-----|------|---------|------|
| **标题** | Inter Bold | Arial Bold | 现代简洁 |

| **代码** | JetBrains Mono | Consolas | 开发者友好 |
| **数字** | Roboto Mono | Courier New | 等宽清晰 |

---

### 4.3 字号规范

| 级别 | 中文 | 英文 | 行高 | 使用场景 |
|-----|------|------|------|---------|
| **H1** | 36px | 32px | 1.2 | 页面大标题 |
| **H2** | 28px | 24px | 1.3 | 章节标题 |
| **H3** | 24px | 20px | 1.4 | 小节标题 |
| **H4** | 20px | 18px | 1.4 | 卡片标题 |
| **正文大** | 18px | 16px | 1.6 | 重要正文 |
| **正文** | 16px | 14px | 1.6 | 标准正文 |
| **正文小** | 14px | 12px | 1.5 | 辅助说明 |
| **辅助** | 12px | 11px | 1.4 | 注释、标签 |

---

### 4.4 字重规范

| 字重 | 名称 | 使用场景 |
|-----|------|---------|
| **100** | Thin | 装饰性文字（少用） |

| **600** | SemiBold | 小标题、标签 |
| **700** | Bold | 主标题、重要强调 |
| **900** | Black | 超大标题（少用） |

### 4.5 字间距与行距

## 七、品牌语调

### 7.1 沟通原则

| 原则 | 说明 | 示例 |
|-----|------|------|
| **专业** | 展现技术实力 | 「采用 K8s 原生架构」 |
| **亲和** | 避免过度技术化 | 「3 分钟创建您的 GPU 实例」 |
| **简洁** | 直截了当 | 「按秒计费，用多少算多少」 |
| **可信** | 数据支撑 | 「SLA 99.5%，故障赔偿」 |

---

### 7.2 文案风格

| 场景 | 风格 | 示例 |
|-----|------|------|
| **官网文案** | 专业 + 简洁 | 「弹性 GPU 算力，5 分钟就绪」 |
| **技术文档** | 准确 + 详细 | 「支持 NVIDIA Device Plugin v0.14.0」 |
| **营销文案** | 吸引 + 行动 | 「立即注册，送 500 元体验金」 |
| **客服沟通** | 耐心 + 解决 | 「您好，请问遇到什么问题？」 |
| **危机公关** | 诚恳 + 担当 | 「我们深表歉意，正在紧急处理」 |

---

### 7.3 禁用表达

| 类型 | 禁用 | 建议 |
|-----|------|------|
| **过度承诺** | 「100% 可用」 | 「SLA 99.5%」 |
| **贬低竞品** | 「比 XX 好 10 倍」 | 「我们提供差异化服务」 |
| **技术黑话** | 「赋能、抓手、闭环」 | 「帮助、工具、完整流程」 |
| **模糊表述** | 「很快、大概」 | 「3 分钟内、约 5 分钟」 |
