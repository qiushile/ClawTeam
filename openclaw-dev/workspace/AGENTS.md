# 研发部专业 Agent 库

你可以根据任务需求，调用以下专业角色来协助完成工作。这些 Agent 详细的指令位于 `workspace/agents/` 目录中。

## 软件架构与设计
- **软件架构师 (software-architect)**: 专注于系统设计、DDD、架构模式和架构决策。
- **后端架构师 (backend-architect)**: 专注于可扩展系统设计、数据库架构和 API 开发。
- **安全工程师 (security-engineer)**: 负责系统安全审计、威胁建模和安全加固。

## 核心开发
- **前端开发人员 (frontend-developer)**: 专注于现代 Web 技术、React/Vue/Angular 和 UI 实现。
- **高级开发人员 (senior-developer)**: 处理复杂逻辑实现、性能优化和技术方案攻坚。
- **移动应用构建器 (mobile-app-builder)**: 专注于原生和跨平台移动应用开发。
- **人工智能工程师 (ai-engineer)**: 负责 AI 模型集成、Prompt 工程和 AI 功能开发。
- **快速原型机 (rapid-prototyper)**: 快速构建 MVP 和概念验证。

## 数据与基础设施
- **数据工程师 (data-engineer)**: 负责数据管道、Lakehouse 架构和可扩展数据基础。
- **数据库优化器 (database-optimizer)**: 专注于模式设计、查询优化和性能调优。
- **自主优化架构师 (autonomous-optimization-architect)**: 持续监控并优化系统性能。
- **AI 数据修复工程师 (ai-data-remediation-engineer)**: 处理数据异常、修复损坏的数据管道。

## 运维与质量
- **DevOps 自动化工具 (devops-automator)**: 负责 CI/CD 流程、基础设施自动化。
- **SRE (sre-site-reliability-engineer)**: 确保系统可用性、监控、SLO/SLI 管理。
- **威胁检测工程师 (threat-detection-engineer)**: 实时监控安全威胁。
- **事件响应指挥官 (incident-response-commander)**: 负责线上故障应急响应。
- **代码审查员 (code-reviewer)**: 确保代码质量和团队标准。
- **Git 工作流大师 (git-workflow-master)**: 维护版本控制规范和流水线。

## 垂直领域与文档
- **微信小程序开发者 (wechat-mini-program-developer)**: 专注于微信生态应用。
- **飞书集成开发人员 (feishu-integration-developer)**: 负责企业协作工具自动化集成。
- **嵌入式固件工程师 (embedded-firmware-engineer)**: 处理底层硬件控制与固件开发。
- **Solidity 智能合约工程师 (solidity-smart-contract-engineer)**: 负责 Web3 与区块链逻辑。
- **技术撰稿人 (technical-writer)**: 编写高质量的 API 文档、README 和技术教程。
