## 四、特殊规则说明

### 4.1 票权规则（可配置）
> 规则选哪个应由各小区业主大会议事规则约定

| 规则类型 | 说明 | 统计方式 |
|----------|------|----------|
| 一户一票 | 每个住户（户主）1票 | 按人数统计 |
| 一面积一票 | 每100㎡计1票，不足100㎡按1票计 | **按面积统计**，选项得票=投票人产权面积之和 |
| 一房一票 | 每个房号1票 | 按房号数统计 |
| 一人一票 | 每个业主1票（一人多房也只算1票） | 按人数统计 |

> ⚠️ **注意**：选择"一面积一票"时，统计维度为面积，各选项得票为投票人产权面积之和，需达到面积2/3门槛才算通过。

### 4.3 参会领票流程
> 参会即计数：点击投票链接后，需先"参会领票"才计入参会人数

1. 业主点击投票链接 → 显示投票说明页面
2. 点击"参会领票"按钮 → 记录参会时间，状态变为"已参会未投票"
3. 领票后跳转投票页面 → 进行投票
4. 提交投票后 → 状态变为"已投票"

### 4.4 参会状态说明

| 状态 | 说明 |
|------|------|
| 未领票 | 点击链接但未点击参会领票 |
| 已参会未投票 | 已点击参会领票，但未提交投票 |
| 已投票 | 已完成投票并提交 |

### 4.5 投票权数与面积统计

### 4.3 认证规则
1. 微信授权获取手机号
2. 业主输入姓名+手机号中间四位
3. 系统比对业主清册，比对成功下发短信验证码
4. 验证成功后完成实名认证
5. 比对失败可申请人工审核

### 4.3 审核流程
1. 业主或业委会发起投票申请
2. 社区进行初审
3. 街道（部门）审核通过后发布
4. 投票进行中
